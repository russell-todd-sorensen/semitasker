ns_purgefiles(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_purgefiles - Purge files by date, keeping max files

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_purgefiles file backupMax

DESCRIPTION

Purge files by date, keeping max files.

COMMANDS

ns_purgefiles file backupMax


file The file parameter is used as a basename to select files to purge. backupMax Specifies how many backup files to keep
