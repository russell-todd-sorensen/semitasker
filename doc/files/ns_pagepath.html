ns_pagepath(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_pagepath - Return path to current virtual server pages

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_pagepath

DESCRIPTION

This is virtual hosting aware command will return absolute path to the directory with pages for the current virtual server

COMMANDS

ns_pagepath

EXAMPLES

 

oss2:nscp 8> ns_pagepath
/usr/local/ns/html

SEE ALSO

nsd
