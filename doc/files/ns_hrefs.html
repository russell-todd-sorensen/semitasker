ns_hrefs(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_hrefs - Returns URLs from anchor elements in a chunk of HTML

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES

SYNOPSIS

ns_hrefs html

DESCRIPTION

This command finds the anchor elements (&lt;A&gt;) in a chunk of HTML, html, and returns a list of the URLs in those elements' href attributes.

COMMANDS

ns_hrefs html

EXAMPLES

 
   % set html {One good website is }
