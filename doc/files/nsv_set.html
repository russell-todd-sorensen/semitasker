nsv_set(n) 4.99 "NaviServer Built-in Commands"

NAME

nsv_set - Set a key value

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO

SYNOPSIS

nsv_set array key value

DESCRIPTION

Set the value for a key in an nsv array. Returns the value the key is set to.

COMMANDS

nsv_set array key value

EXAMPLES

 
   % nsv_set shared_array key1 value1
   value1
