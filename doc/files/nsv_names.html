NAME

nsv_names - List the nsvs in use

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO

SYNOPSIS

nsv_names ?pattern?

DESCRIPTION

Return a list of all the nsvs in use, optionally only those matching pattern. If no matching nsvs are in use returns the empty string.

COMMANDS

nsv_names ?pattern?

EXAMPLES

 
   % nsv_names
   shared_array
