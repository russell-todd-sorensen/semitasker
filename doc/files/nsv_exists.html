nsv_exists(n) 4.99 "NaviServer Built-in Commands"

NAME

nsv_exists - Test whether a key exists

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO

SYNOPSIS

nsv_exists array key

DESCRIPTION

Test whether a key exists in the nsv array.

COMMANDS

nsv_exists array key

EXAMPLES

 
   % nsv_exists shared_array key1
   1

the key exists

 
   % nsv_exists shared_array key2
   0

the key does not exist
