ns_adp_trunc(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_adp_trunc - Truncate the ADP output buffer

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_adp_trunc

DESCRIPTION

This command truncates the ADP output buffer. Any data that was previously queued for sending using ns_adp_puts or other means is cleared from the buffer and will not be sent.

COMMANDS

ns_adp_trunc
