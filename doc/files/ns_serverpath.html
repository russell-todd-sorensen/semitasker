ns_serverpath(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_serverpath - Returns absolute path to current virtual server home

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_serverpath

DESCRIPTION

This is virtual hosting aware command that returns absolute path to the server home for current virtual server context

COMMANDS

ns_serverpath

EXAMPLES

 

oss2:nscp 6> ns_serverpath
/usr/local/ns
