ns_reinit(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_reinit - Reinitialize the current interpreter

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_reinit

DESCRIPTION

This command reinitializes the current interpreter by calling ns_cleanup then ns_init.

COMMANDS

ns_reinit

SEE ALSO

nsd

KEYWORDS

ns_cleanup , ns_ictl , ns_init , ns_markfordelete
