ns_shortcut_filter(n) 4.99 "NaviServer Built-in Commands"

NAME

ns_shortcut_filter - Stop processing url filters

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    SEE ALSO
    KEYWORDS

SYNOPSIS

ns_shortcut_filter when method urlPattern

DESCRIPTION

Other filters that also match when+method+urlPattern will not run after this filter.

COMMANDS

ns_shortcut_filter when method urlPattern

SEE ALSO

ns_schedule_filter, nsd
