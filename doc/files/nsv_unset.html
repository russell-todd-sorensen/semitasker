nsv_unset(n) 4.99 "NaviServer Built-in Commands"

NAME

nsv_unset - Unset an array or a key

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    COMMANDS
    EXAMPLES
    SEE ALSO

SYNOPSIS

nsv_unset array ?key?

DESCRIPTION

Unset an array or a single key from an array. If successful returns an empty string.

COMMANDS

nsv_unset array ?key?

EXAMPLES

 
   % nsv_unset shared_array key1

unset a key

 
   %nsv_unset shared_array

unset an array
