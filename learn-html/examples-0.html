This is some text in a paragraph tag.

 /* example 2 */ .p { color: white; background-color: tan; width: 300px; margin: 10px; padding: 15px; } /* end example 2 */

The paragraph tag is a block level tag. Each paragraph begins and ends with a new line.

 /* example 3 */ .bold { font-size: larger; font-weight: bold; } /* end example 3 */

Some paragraphs might need special formatting.

 /* example 4 */ .reddish { color: rgba(255,0,0,.85); }

CSS is useful for getting the effect that you want.

CSS stands for Cascading Style Sheets
