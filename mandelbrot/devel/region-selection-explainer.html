The above html represents the main selection tool for fractal explorer:
  1. An outer container setting the relative position of:
  2. A canvas with fractal image,
  3. A dashed-edge selector which outlines the selection, and
  4. An upper mouse-event capture surface which captures mousedown, mousedrag and mouseup events.
The canvas and event capture surface have the same dimensions and border. The border is an important element as it allows selecting the entire canvas, or any edge of corner without losing contact with the event surface.
