Chapter 3: Template Engine Feature Comparison
Standalone Does not require installation of Scripting Language to compile templates.
Syntax Validation Template engine can detect syntax errors in templates.
Variable Safety Extent of access to and control over local, context and system variables.
Command Safety Extent of access to scripting language commands within template.
Caching Extent of support for reuse of template code and/or compiled templates.
Shared Setup Ability to share code which sets up data and resources.
Template Security Extent to which the template engine allows, or is unable to prevent arbitrary code execution or variable access.
Easy to Use Extent to which template language simplifies task of merging data with presentation.
Variable Modifiers Extent of support for formatting variables for display.
Template Functions Extent of built in functions for specialized formatting.
Filters Extent of ability to pre- or post- process a template.
Resources Ability to add specialized data and features to a template.
Add-ons Ability to reuse code from external sources.
Plugins Ability to add library code from external sources.
Debugging Ability to quickly analyze each stage of template engine operation and template compilation to locate and fix template errors.
Compiling Ability to detect and recompile templates which have changed.
Performance Design features which enhance rapid development of bug free and fast applications.
