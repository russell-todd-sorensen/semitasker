Chapter 02: Quickstart Tutorial

One Tic-Tac-Toe design goal is to avoid creating a new programming language. Instead, Tic-Tac-Toe relies on the simple structure of the Tcl language as a starting point.

For example in Tcl, the set command is used to set the value of a variable. In a Tcl script this looks like:

 set variableName value

In a Tic-Tac-Toe template you set a variable like this:

[set variableName value]
