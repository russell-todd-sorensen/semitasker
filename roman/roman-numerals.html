  • Unicode Symbols for Large Roman Numerals
    • A few years back I was looking at some unicode tables an out popped something really new to me: Roman numerals with larger values. Using these I got a domain name to use in testing something called punycode to represent International Domain Names with non-ascii characters. So, okay the AKC didn't think ahead very far with their naming scheme, but how 'bout those engineering the internet with only ASCII? So the larger Roman Numerals I found are these:
    • ↀ = 1000 (C,D alt of M),
    • ↁ = 5000,
    • ↂ = 10,000,
    • ↇ = 50,000 and
    • ↈ = 100,000.
    I used these to create a domain name for testing purposes:
    • ↈↀↁↇ = 144000 = 100,000+(50,000-5000-1000).
    • I also found the rules for constructing larger numbers inconsistent, so I used the "shortest is best" rule, alternatives are these:
    • ↀↈↁↇ = 100,000-1000+(50000-5000) and
    • ↈↂↂↂↂↀↀↀↀ = 100,000+4*10,000+4*1000.
